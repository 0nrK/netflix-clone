## Netflix CLONE

A Netflix clone that build with React.js on frontend.

You can view live site [Right Here](https://reactnetflixclonex.herokuapp.com/)

## Built with
- React.js
- React-router-dom for routing between pages
- Tailwind CSS
- Flexbox 
- Responsive Design

## In the project directory, you can run:

```
npm start
```

Open http://localhost:3000 to view it in the browser.
